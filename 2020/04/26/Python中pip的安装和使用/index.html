使用pip对所有可更新包进行更新

什么是 pip

pip 是 Python 包管理工具，该工具提供了对Python 包的查找、下载、安装、卸载的功能。

目前如果你在 python.org 下载最新版本的安装包，则是已经自带了该工具。

你可以通过以下命令来判断是否已安装。

curl https://bootstrap.pypa.io/get-pip.py -o get-pip.py
python get-pip.py

如果是使用 python3则替换为

python3 get-pip.py
